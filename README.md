# 📚 StudySync – Online Learning Platform

StudySync is a modern online learning platform built using **PHP, MySQL, HTML5, CSS3, and JavaScript**. It allows students to browse courses, enroll in them, manage their learning, and contact the platform. An administrator can manage the complete website through a secure admin panel.

# ⚙ System Requirements

Install the following software before running the project.

- XAMPP
- Apache
- MySQL
- PHP 8.x or above

---

# 🚀 Installation Guide

## Step 1

Download or clone this repository.

```
git clone <repository-url>
```

or download the ZIP file and extract it.

---

## Step 2

Move the project folder into the XAMPP htdocs directory.

Example:

```
C:\xampp\htdocs\StudySync
```

---

## Step 3

Open **XAMPP Control Panel**.

Start the following services:

- Apache
- MySQL

---

## Step 4

Open phpMyAdmin.

```
http://localhost/phpmyadmin
```

---

## Step 5

Create a new database.

Database Name:

```
studysync
```

---

## Step 6

Import the SQL file.

Location:

```
database/studysync.sql
```

Click

Import → Choose File → studysync.sql → Go

---

## Step 7

Open the website.

```
http://localhost/StudySync/
```
